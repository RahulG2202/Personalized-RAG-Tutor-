# RAG Tutor AI Frontend

Next.js, Tailwind CSS, and GSAP frontend for the personalized RAG tutor.

## Setup

```bash
npm install
npm run dev
```

The app expects the backend at `http://localhost:8000` by default. To override it, create `.env.local`:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```
